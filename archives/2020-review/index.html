2020 年度总结

AnnualReview No Reply

0x00 前言

“惊喜不断” 的 2020 终于快结束啦，在年初疫情的影响下假期高强度加长，我也做了不少有意思的事，在年终总结一下吧
因为这篇博客是风鸟在写，所以你不太可能会在这份年度总结中读到和线下生活相关的内容（

0x01 今年玩的游戏

熟悉我的人应该都知道我有严重的 Motion Sick，大部分 3D 游戏都撑不过十分钟，于是今年也买了不少游戏玩了一会就放库里吃灰了 这就是我买的VN也吃灰的理由
简单说说今年投入了不少时间的游戏吧，说起这个当然得先吹 NieR: Automata
今年终于实现了全 100% 三周目完美毕业，可喜可贺，可喜可贺

接下来就是 Crystar 啦，这游戏虽然什么都好但就是不好玩，总体感觉还是制作很不错的游戏，但玩法太单一和强行多周目很让人头大

另一个游戏是 Code Vein，虽然在手感和剧情重复的机制上有一些问题，总体上还是很棒的
目前刚刚开始二周目，计划在 2021 年把 100% 拿到手

还有不得不提的 Black Mesa，HL1 的重制版，大家都懂
不过返校后基本没时间再碰了，目前也没达到全成就

今年 Halo MCC 终于更新完啦，光环的故事在 PC 上也走向了尾声，成就估计这辈子是刷不完了

接下来就是某个用黑黄公告跳票了三次的游戏了，当初画的饼真的没几个做出来的… 虽然游戏整体也不错，但画饼不填坑还是很不好的

除了上面提到的游戏，令人印象深刻的还有一些 NSFW VNs，这里就不多提了（
如果你有关注我的频道，基本上每次我玩 VN 之前都会把游戏拆一遍然后在频道里水一下，看看记录大概就知道我玩了啥

0x02 今年挖的大坑

今年创建了 Berd’s Playground，从此开始了一去不复返的水频道生活（
当然在学习和摸鱼之余我还是有认真做一点东西的.jpg

BerdRefactorFramework

这是一个 JS 全自动反混淆框架
最初只是被群友勾起好奇心去做的，到现在已经可以全自动拆 obfuscator.io，sojson 以及部分 jscrambler 了
当然是仅限 iL 成员使用的，目前还没有公开

SakuraFrp [LINK]

一个 iDea Leaper 接手的项目 月月亏损，全靠群富豪撑着没倒闭
这应该是目前维护的最主要的项目之一了，优先级还蛮高的（

SakuraFrp Launcher [LINK]

SakuraFrp 的启动器，类似于客户端一样的东西，不必多说
写这个学了不少 IPC 和 Windows 服务相关的知识，还有一些神秘的 Windows 特性

Nikki4 Utils

今年开的新坑，目标是整一个闪暖服务器或者协议 Hack 衣服太贵了买不起而且有的还不复刻
协议已经拆完了，也有做出 POC，但是实在是没动力去写，这游戏的小玩意多到爆炸，光是一个登录就有上百个 Req 和 Noti
虽然是主要项目但不知道要咕到什么时候

MoeSearch [LINK]

这是从 2018.10 就开始搞的老坑了，大概每年只都推一点进度
今年主要搞了网站部分，新的索引程序 (MoeSearchLIRE / MoeSearchIndexer)，一个 Bot (@moesearch_bot) 和各种玄学优化
本来计划写完就开源的但是到现在还没有写完
遗憾的是，在 2020.7.29 我的两块 12T 硬盘都满了，早苗因为疫情原因回不去，一直暂停索引到现在

两个隐藏项目

说出来肯定会炸表，是不宜描述的东西~

0x03 今年做的零碎小玩意

这些小玩意大部分都被我放到频道里了，没放到 GitHub

Covid-19.WTF

这是在年初给大家提供疫情信息聚合 & Covid-19 速报频道的反代
一共用了两个域名: ncov.wtf (已被 DNS 污染) 以及 covid-19.wtf
国内抗疫还是非常强的，年中就已经关站啦

奇怪的瘟疫公司 MOD [LINK]

年初疫情严重各种魔幻新闻冒出来时做的 (草)

Portal Stories: Mel 的多核兼容性补丁 [LINK]

Portal 2 其实已经修复过这个问题了，但 Mel 似乎并没有跟进引擎版本，所以在超过 28 线程的机器上跑不起来，弄了这个小补丁来解决问题

BerdSocks

这是个兼容 VMess / SS(R) / 自有协议的纯 C# 实现
基本功能都写完了，学到了不少东西 (特别是密码学的怪知识)，不过读 SSR 代码的过程很让人难受，写的实在是太 [消音] 了
最后当然弃坑啦，因为没人用 我自己都不用（

Chromium 的 Flash 补丁 [LINK]

假期上网课的时候有一些古董网站用到了 Flash (点名批评 MOOC)，都 0202 年了让我上哪找支持 Flash 的浏览器啊
最后只好弄个补丁编译了专用的浏览器来解决这问题

给 Chromium 加上了 GMTLS 和相关套件的支持

这其实是个挺复杂的项目，从啃国密标准到亲手实现 SM2，SM3，SM4 并且最后实现 GMTLS，还是踩了不少坑，强行学了一堆密码学知识 虽然都快忘光了
当然这只是一个学习性的项目，没有发布任何代码或者 Binary

Mages Tools [LINK]

最开始我只是想玩 Steins; Gate LBP 和 MDE 而已…
结果这俩游戏的 Steam 版本居然没汉化，真够急人的
凑合着去玩那些移植版可不是我的风格，直接拆游戏移植文本，顺便把工具丢去 GitHub 了
相关链接: LBP 汉化补丁的发布贴，MDE汉化补丁发布贴

Namecheap Panel [LINK]

顾名思义，一个管理 Namecheap 域名的小玩意
他们的 Portal 实在是太太太太太卡了，受不了，最后重新写了这个面板来管理域名（虽然大部分 DNS 都迁去 Cloudflare 了
设计的时候考虑到了各种安全问题，因此把所有操作都放到前端进行并且让用户可以自定义中间代理的 URL
相关链接: 可以用的在线版本

Yu-RIS Tools [LINK]

又是为了打游戏整出来的项目（
这次是 MaggotBaits，发现游戏没有汉化后连夜拆游戏拆文本写翻译工具，成功实现了拆包，解密，重新封装文本之类的操作
然而因为我的 N95 日语无限期咕咕中，现在还没玩上

LC-ScriptEngine Tools [LINK]

同样是因为我想玩 VN 搞出来的，这次是黑兽1 (旧版)，实际上这个并不能算工具集，因为目前只有一个打包解包工具
拿到游戏发现 VT 接近一半的报毒，怪吓人的，手动脱壳杀毒然后拆了游戏重新封包
虽然处理好了但我到现在都没玩上这游戏，太忙了

xLua Dumper [LINK]

又一个一时兴起搞的小玩具，在拆某 “国产3A” 时的副产物
详情在我 另一篇博客 中有提到，此处不再赘述

EmotivLicenseServer

假期在家闲得慌怎么办？掏出 EPOC+ 尝试玩 EEG 呗
然而 Emotiv 的离谱定价让我没办法用上这个 Headset 的完整功能，一气之下就把他们的协议拆了，整了个授权服务器 最后还是因为穷
这种项目当然是不能公开源码或者 Binary 的（

SEU DailyReport [LINK]

疫情期间用来体温打卡的小玩意，形式主义好.jpg
上网搜了一圈发现现成的都只有 Tampermonkey 脚本，就自己整了个全自动的上报

0x04 今年的奇怪收藏

假期在家里蹲太久了还是挺麻烦的，所以我开始收收藏里的一些老番的 BD 和手办 烧钱
这些奇怪的收藏就不一个个列举了（我也不太记得具体有啥，都放在家里），在频道里搜 #好耶 这个 Tag 就能看到一部分

值得一提的是 Violet Evergarden 的 BD 收纳盒真的让我找的好辛苦，一个盒子卖到 3000 多实在是太离谱了，最后没办法只好自己做了一个

看最终成品图做的很轻松简洁，实际上我花了超多时间去对着一些零碎的晒图测量比例、元素位置然后各种渠道挖配色、素材以及想办法把各种素材拼起来，最后才得到上面这个 PSD

打样然后不停修改了两三次花了不少钱，但最后的效果也令人满意，做出来和官方收纳盒几乎没有任何差别

0x05 最后来点好玩的

零零碎碎写了一堆，基本上把今年干的事情概括了个七七八八
每年跨年好像大家都在出题，今年我也来整一个（

    示例: https://berd.moe/2020/c542d979-fe16-49f7-83a6-6f1e14524801
